      • Code.zip (Python w/ Tensorflow)
      • Code & Datasets (zip file size 60MB, redirected to Dropbox)
      • filter_ui.py Script to filter users and items that have too few interactions. In our experimental dataset, min_user=3 and min_item=10. Note that min_user must be no less than 3, since we need to have at least one example for train, valid, and test respectively for each user. Threshold for min_item is more flexible which can be used to control the sparsity of the interaction matrix.
